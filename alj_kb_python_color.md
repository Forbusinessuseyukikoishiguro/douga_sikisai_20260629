# 【新人エンジニア向け】AIキャラクターの「色」をPythonで数値にする方法

## この記事でできるようになること

AIで生成したキャラクター画像から、「この色のコードって何番？」をPythonで調べられるようになります。

**こんな場面で使えます：**

- 白うさぎキャラのクリーム色のベスト → 正確な色コードを記録しておきたい
- 再生成したら微妙に色が変わってしまった → 元の色と数値で比較したい
- Live2DやCanvaで同じ色を使いたい → RGBやHEXコードが必要

---

## 準備：ライブラリをインストールする

ターミナル（コマンドプロンプト）で以下を実行します。

```bash
pip install opencv-python numpy
```

| ライブラリ名 | 何をする道具？ |
|---|---|
| opencv-python | 画像を読み込んだり、ピクセル（点）の情報を取り出す |
| numpy | 数値データを配列（表）として扱う |

> **ライブラリとは？**  
> Pythonに最初から入っていない「追加の道具箱」です。インストールすることで使えるようになります。

---

## コード全文

```python
import cv2
import numpy as np

image_path = "your_character.png"
image = cv2.imread(image_path)

if image is None:
    print("画像を読み込めませんでした。ファイルパスを確認してください。")
else:
    height, width, channels = image.shape

    print(f"幅: {width} px")
    print(f"高さ: {height} px")
    print(f"チャンネル数: {channels}")

    pixel_top_left = image[0, 0]
    print(f"左上のピクセル (B, G, R): {pixel_top_left}")

    center_y, center_x = height // 2, width // 2
    pixel_center = image[center_y, center_x]

    b, g, r = pixel_center
    hex_color = "#{:02X}{:02X}{:02X}".format(r, g, b)
    print(f"中央のHEXカラーコード: {hex_color}")
```

---

## 一行ずつ解説

### ① ライブラリを呼び出す

```python
import cv2
import numpy as np
```

`import` は「道具箱を開ける」イメージです。  
`as np` は長い名前を短く呼ぶための別名です。毎回 `numpy.〇〇` と書かなくて済むよう、`np.〇〇` と省略できるようにしています。

---

### ② 画像ファイルのパスを指定する

```python
image_path = "your_character.png"
```

`"your_character.png"` の部分を、自分の画像のファイル名に書き換えてください。  
Pythonスクリプトと同じフォルダに画像を置いておくのが一番楽です。

---

### ③ 画像を読み込む

```python
image = cv2.imread(image_path)
```

`cv2.imread()` は「画像ファイルを読み込む」関数です。  
読み込んだ結果（ピクセルデータの集まり）を `image` という変数に入れています。  
中身は「数値の表（NumPy配列）」になっており、全ピクセルの色情報が数字として格納されています。

---

### ④ 読み込みに失敗したときの処理

```python
if image is None:
    print("画像を読み込めませんでした。ファイルパスを確認してください。")
```

`cv2.imread()` はファイルが見つからないとき `None`（何もない）を返します。  
ファイル名のスペルミスや、ファイルの場所が違う場合にここに入ってきます。

---

### ⑤ 画像のサイズを取り出す

```python
height, width, channels = image.shape
```

`.shape` は「この画像の形（サイズ）を教えて」と聞くプロパティです。  
返ってくる値は `(高さ, 幅, チャンネル数)` の3つで、左から順に変数に入ります。

> **チャンネル数とは？**  
> 色の情報の種類数です。カラー画像なら `3`（青・緑・赤）、透明情報つきなら `4` になります。

```python
print(f"幅: {width} px")
print(f"高さ: {height} px")
print(f"チャンネル数: {channels}")
```

`f"..."` は **f文字列** と呼ばれる書き方で、`{}` の中に変数を入れると値が表示されます。  
たとえば `width` が `512` なら `幅: 512 px` と表示されます。

---

### ⑥ 特定の座標のピクセルを取り出す

```python
pixel_top_left = image[0, 0]
```

`image[y座標, x座標]` という書き方で、特定のピクセルの色情報を取り出せます。  
`[0, 0]` は「縦0番目・横0番目」= 画像の左上の角のピクセルです。

> **注意：** NumPy配列は `[行番号, 列番号]` = `[y, x]` の順です。数学のグラフ（x, y）と逆順なので最初は混乱しやすいです。`image[y, x]` と覚えておきましょう。

取り出したピクセルの値は `[数値, 数値, 数値]` の形で出てきます。これが**青・緑・赤それぞれの強さ（0〜255）**です。

---

### ⑦ 画像の中央のピクセルを調べる

```python
center_y, center_x = height // 2, width // 2
pixel_center = image[center_y, center_x]
```

`//` は整数の割り算です（小数点以下を切り捨て）。  
たとえば高さ512なら `512 // 2 = 256`、つまりy座標256が中央です。

---

### ⑧ BGRをRGBに変換してHEXコードにする

```python
b, g, r = pixel_center
hex_color = "#{:02X}{:02X}{:02X}".format(r, g, b)
```

OpenCVは色を **B（青）→ G（緑）→ R（赤）** の順で格納しています。  
一般的なRGBとは逆順なので要注意です。ここで3つの値をそれぞれ `b`・`g`・`r` という変数に分けています。

`{:02X}` は数値を16進数の2桁で表示する書き方です。たとえば `255` は `FF`、`0` は `00` になります。HTMLのカラーコードの形式がこれです。

---

## 実行結果のイメージ

うまくいくと、こんな感じで表示されます。

```
幅: 512 px
高さ: 512 px
チャンネル数: 3
左上のピクセル (B, G, R): [240 235 220]
中央のHEXカラーコード: #ECEBE0
```

この `#ECEBE0` をCanvaやFigmaのカラー入力欄にそのまま貼り付ければ、ピッタリ同じ色が再現できます。

---

## よくあるエラーと対処法

| エラーのメッセージ | 原因 | 対処法 |
|---|---|---|
| 画像を読み込めませんでした | ファイル名かパスが違う | ファイル名のスペルを確認。スクリプトと同じフォルダに画像を置く |
| `ModuleNotFoundError: No module named 'cv2'` | OpenCVがインストールされていない | `pip install opencv-python` を実行する |
| 色の値が思ったより違う | JPGは保存時に圧縮で色が変わる | なるべくPNG形式の画像を使う |

## まとめ

| コード | 何をしているか |
|---|---|
| `import cv2` | 画像処理ライブラリを呼び出す |
| `cv2.imread()` | 画像を読み込む（結果は数値の表） |
| `image.shape` | サイズ（高さ・幅・チャンネル数）を取得 |
| `image[y, x]` | 特定座標のピクセル（色情報）を取り出す |
| `b, g, r = pixel` | BGRの値を変数に分ける（OpenCVはBGR順） |
| `"#{:02X}{:02X}{:02X}"` | RGBをHEXカラーコードに変換する |

**「画像 = 数値の表」** というイメージがつかめると、画像処理の世界がぐっと近くなります。
