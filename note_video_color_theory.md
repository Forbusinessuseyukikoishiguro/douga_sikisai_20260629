# 【IT研修講師向け】動画解説で「伝わる色」を選ぶための色彩設定入門

## はじめに：なぜ研修動画に「色の知識」が必要なの？

スライドや解説動画を作るとき、「なんとなくそれっぽい色を使っていた」という経験はないでしょうか。

色には、**見る人の注意・理解・感情に直接影響する力**があります。研修動画では「わかりやすさ」が最優先ですが、色の選び方を間違えると、どれだけ内容が良くてもスライドが「読みにくい」「なんか信頼感がない」「疲れる」と感じさせてしまいます。

この記事では、動画解説を初めて作る研修講師の方向けに、**色彩の基本理論と、研修動画で使える配色ルール**をまとめます。

---

## 1. 色の三属性を押さえる

色を語るうえで欠かせない基礎知識が、色の**三属性**です。

| 属性 | 意味 | 研修動画での影響 |
|---|---|---|
| **色相（Hue）** | 赤・青・緑などの「色の種類」 | テーマカラーの印象を左右する |
| **明度（Value）** | 明るさの度合い | 文字の読みやすさに直結する |
| **彩度（Saturation）** | 色の鮮やかさ | 高すぎると目が疲れる |

研修動画でよくある失敗は「彩度が高すぎる色をベースに使ってしまう」ことです。鮮やかすぎる背景や文字色は、長時間視聴で受講者の目を疲弊させます。

> **基本方針：** ベース色は彩度を落とし、強調箇所だけ彩度を上げる。

---

## 2. 色彩心理：色が与える印象

色は見る人に無意識の印象を与えます。研修テーマや対象受講者に合わせて選ぶと、動画全体の「トーン」が整います。

| 色 | 与える印象 | 研修での活用例 |
|---|---|---|
| 青 | 信頼・冷静・知性 | IT技術・セキュリティ・システム系 |
| 緑 | 安心・成長・安定 | 新人研修・コミュニケーション系 |
| オレンジ | 活気・親しみやすさ | ワークショップ・演習パート |
| 赤 | 注意・緊急・重要 | 警告・エラー・禁止事項の強調 |
| 紫 | 高品質・専門性 | 上級者向け・資格試験対策 |
| グレー | 中立・落ち着き | テキスト・補足説明・背景 |

研修動画は「学習コンテンツ」なので、**青・緑系をベースに置くと落ち着いた信頼感**が生まれやすいです。赤はあくまで強調用に限定して使うのが原則です。

---

## 3. 配色の基本ルール：3色ルール

動画スライドで使う色は、原則として**3色以内**に絞ります。

```
ベースカラー（70%）：背景・大きな余白
メインカラー（25%）：見出し・重要テキスト
アクセントカラー（5%）：強調・ボタン・矢印
```

### 色数を絞る理由

- 色が多いと「どこが重要か」が伝わらなくなる
- 視線が分散し、学習効率が下がる
- スライド全体の統一感がなくなる

> **よくある失敗例：** 見出しは青、本文は黒、強調は赤、コードは緑、注釈は紫……と色を増やしてしまうケース。受講者は「この色の違いに意味があるのか？」と迷います。

---

## 4. コントラスト比：「読めるか」を数値で確認する

どれだけ色のセンスがよくても、**文字と背景のコントラスト比が低いと読めません**。

WCAG（Webコンテンツアクセシビリティガイドライン）では、通常テキストのコントラスト比は **4.5:1 以上** が推奨されています。研修動画でも同じ基準を目安にすると安全です。

### 代表的な組み合わせの目安

| 組み合わせ | 可読性 |
|---|---|
| 白背景 × 黒文字 | 最も高い（21:1） |
| 白背景 × 濃紺文字 | 高い（推奨） |
| 白背景 × 明るい黄色文字 | 低い（NG） |
| 黒背景 × 白文字 | 高いが長時間は疲れやすい |
| グレー背景 × グレー文字 | 低い（NG） |

コントラスト比は無料ツール「**WebAIM Contrast Checker**」で確認できます。HEXカラーコードを入力するだけで即座に判定できるので、スライド制作前にチェックする習慣をつけましょう。

## 5. 研修動画に向いている配色パターン

### パターンA：ライトテーマ（初心者・新人研修向け）

```
ベース：#FFFFFF（白）または #F5F5F5（薄いグレー）
メイン：#1A4A7A（濃紺）
アクセント：#E07B39（オレンジ）
```

明るく親しみやすい印象。視認性が高く、長時間視聴でも目が疲れにくい。

---

### パターンB：ダークテーマ（ターミナル・コード解説向け）

```
ベース：#1E1E1E（ダークグレー）
メイン：#D4D4D4（明るいグレー）
アクセント：#4EC9B0（ティール）
```

コマンドラインやコードエディタの画面と自然に馴染む。エンジニア向け研修で多用される配色。

---

### パターンC：ナチュラルテーマ（ソフトスキル・コミュニケーション研修向け）

```
ベース：#FAFAF5（オフホワイト）
メイン：#2E6B4E（深緑）
アクセント：#C9973A（ゴールド）
```

落ち着きと温かみのある配色。ハラスメント研修・チームビルディング・EQ研修など、人間関係を扱うテーマに合う。

---

## 6. 色を「役割」で使い分ける

研修動画では、色を**感覚ではなく役割**で割り当てるのが大切です。決めたルールを動画全編で一貫させることで、受講者は「この色 = この意味」と無意識に学習します。

| 役割 | 色の例 | 使う場面 |
|---|---|---|
| 情報（neutral） | グレー・濃紺 | 通常テキスト・本文 |
| 強調（emphasis） | オレンジ・黄色 | キーワード・重要ポイント |
| 警告（warning） | 赤・赤橙 | エラー・注意事項・禁止 |
| 成功（success） | 緑 | 正解・完了・OK |
| コード（code） | 背景：ダークグレー、文字：ライトグリーン | コードブロック |

---

## 7. やってはいけない色使い5選

研修動画でよく見かける、避けるべき色使いをまとめます。

1. **赤と緑の組み合わせのみで情報を区別する**  
   → 色覚多様性（赤緑色盲）のある受講者に伝わらない。形・文字・アイコンと組み合わせる。

2. **明るい色同士を並べる**  
   → 黄色背景に白文字など。コントラスト不足で読めない。

3. **スライドごとに違う色で同じ要素を表す**  
   → 「見出しが青のページと赤のページが混在」など。受講者が混乱する。

4. **強調のつもりで文字色を7色使う**  
   → 強調しすぎてどこも強調されていない状態になる。

5. **背景に高彩度の色を使う**  
   → 鮮やかな青や緑の背景は視聴疲れを招く。背景は必ず低彩度・低明度差に。

---

## まとめ

| ポイント | 内容 |
|---|---|
| 三属性を理解する | 色相・明度・彩度の役割を把握する |
| 色彩心理を活用する | テーマに合った色で受講者の印象をコントロールする |
| 3色ルールを守る | ベース70%・メイン25%・アクセント5% |
| コントラスト比を確認する | 4.5:1以上を目安にWCAM Checkerで確認 |
| 色に役割を持たせる | 決めたルールを動画全編で一貫させる |

色の選び方に「正解」は一つではありませんが、**理由のある色選び**ができるようになると、動画全体のクオリティが一段上がります。まずは「3色以内」「コントラスト比の確認」の2点から意識してみてください。
